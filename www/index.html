WSIC - An open source, efficient and minimalistic web server

Developed by students mastering in Computer Security at BTH, Sweden for the course DV1457 - Programming in Unix.

WSIC is open source and available for free at https://github.com/AlexGustafsson/wsic.

Features

  • Amazingly fast with a fully multi-threaded, event-driven architecture
  • Minimalistic configuration
  • Support for CGI
  • Supports TLS 1.2 and TLS 1.3
  • Hardened for modern security needs

Getting started

Getting started with WSIC couldn't be easier. Just run docker run -it -p8080:80 wsic/wsic.

Seeing this page on your own domain? Read about configuring WSIC in the section below.

Configuration

WSIC uses a simple TOML-based configuration file. An example config file looks like this:

[server]
daemon = false

[servers]
[servers.test]
domain = "localhost"
rootDirectory = "www"
port = 8080
[servers.testSecure]
domain = "localhost"
rootDirectory = "www"
port = 8443
certificate = "server.cert"
privateKey = "server.key"
ellipticCurves = "P-384:P-521"
validateCertificate = false
dhparams = "dhparams.pem"
